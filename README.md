## 🚀 Como Configurar o Time de Super Skills

### Passo 1: Crie um novo Projeto no Claude

1. Acesse [claude.ai](https://claude.ai)
2. Clique em **Projects** na barra lateral esquerda
3. Clique em **+ Create Project**
4. Dê um nome como **Time de Super Skills** ou **Marketing Advisory Board** (fica à sua escolha)

### Passo 2: Adicione as Skills

1. Dentro do seu novo projeto, clique em **Add Skills** (ou no ícone `+` na seção de Skills)
2. Clique em **Upload Skill**
3. Selecione o arquivo `.md` da skill desejada

> ✅ Seu projeto agora deve mostrar **1 skill instalada**.

### Passo 3: Comece a Usar

Abra um novo chat dentro do seu projeto. As skills **ativam automaticamente** com base no que você pergunta.

## 🧩 O arquivo `skill.md` — O Cérebro do Sistema

O arquivo `skill.md` é o **arquivo principal** que conecta todas as skills e dá contexto ao Claude sobre como agir como um time de consultores especialistas.

### Como instalar:

1. Acesse a página **The Skill.MD File (The Brain)** no Notion
2. Clique com o botão direito no arquivo `skill.md`
3. Selecione **"Download"**
4. Vá ao seu projeto no Claude
5. Clique em **Add Skills** → **Upload Skill**
6. Selecione o arquivo `skill.md` baixado

> 💡 **Dica:** Instale o `skill.md` como a **primeira skill** do projeto. Ele funciona como o sistema operacional das outras skills.

## 💡 Dicas de Uso

### Como as Skills se ativam?
As skills **detectam automaticamente** o contexto da sua pergunta. Quando você pergunta sobre criação de ofertas, o Hormozi ativa. Quando fala de conteúdo, o Gary V entra em cena.

### Posso usar múltiplas skills no mesmo chat?
Sim! O Claude consegue **combinar** as perspectivas de múltiplos consultores em uma única resposta. Exemplo:
> "Com a perspectiva do Hormozi na oferta e do Brunson no funil, crie minha estratégia de lançamento"

### Quantas skills posso ter em um projeto?
O Claude suporta **múltiplas skills por projeto**. Recomenda-se começar com o `skill.md` + 3 a 5 skills principais e ir adicionando conforme a necessidade.

### Como atualizar uma skill?
Basta fazer o upload de uma nova versão do arquivo `.md` e remover a antiga no painel de skills do projeto.
